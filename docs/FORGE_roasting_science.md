## 4. Core Roasting Philosophy — Rao Declining RoR

All profiles follow the Scott Rao declining Rate of Rise architecture:

- **Charge hot** — open with high heat to build thermal momentum through drying
- **Steadily decrease heat** — after Maillard begins, heat only goes down or holds. Never increases.
- **RoR peaks mid-Maillard** (~4:00–5:00) then slopes continuously downward to first crack
- **Arrive at FC on residual energy** — heat should already be low before first crack begins
- **At first crack, drop heat hard** — the bean's own exothermic reaction carries development
- **A single heat re-increase = RoR spike = flat, baked, or harsh cup**

### Reference Curve Targets (Filter Roast Profile Chart)
- Filter A (cyan) — FC ~6:40, DTR ~10% — closest analog to Casey's machine for standard roasts
- Filter B (yellow-green) — FC ~7:00 — target for denser naturals
- Filter D (green) — FC ~8:00 — target for washed Full City+ Rao style

### Process Type → Curve Assignment
| Process | Target Curve | Notes |
|---------|-------------|-------|
| Anaerobic Washed | Steep fast decline | Shorter roast, juicy fuller body |
| Washed | Filter A/B moderate decline | Longer roast breaks down acidity, sweetness shines |
| Lactic Natural | Moderate decline | |
| Standard Natural | Filter A slow decline | |
| Ethiopian Natural | Filter A/B gentle | Preserve florals — earliest H:5 entry |

## 9. General Roasting Rules (Never Violate)

1. Heat only decreases or holds after Maillard begins — never increases
2. Natural process beans — trust sound and smell over color
3. Scratch test every natural process roast — surface color lies
4. Clear chaff collector before every natural process roast
5. Do not use a single sense alone — use all five together
6. Log every roast — FC time, dev time, total time, batch weight, cup notes
7. Weigh green before and roasted after — 11–13% weight loss = light roast check
8. Independent cooling in colander with fan = better preservation of light roast characteristics than built-in cool cycle
